Lista de Descrição

While
O comando while permite que um certo trecho de programa seja executado ENQUANTO uma certa condição for verdadeira.
Do While
O comando do-while permite que um certo trecho de programa sera executado ENQUANTO uma certa condição for verdadeira
For
O comando for permite que um certo trecho de programa seja executado um determinado número de vezes.
Break
O comando break encerra o laço no ponto em que for executado.
Continue
O comando continue desvia o fluxo para o início do laço.
